# Bot SMS Telegram - Deploy Render.com

Bot 100% funcional rodando 24/7 no Render.com

## Deploy Automático

Este bot está configurado para deploy no Render.com (free tier).

## Variáveis de Ambiente

Configure no Render.com:
- `TELEGRAM_TOKEN`: Token do bot
- `SMS_ACTIVATE_API_KEY`: Chave SMS Activate
- `PLUGGY_CLIENT_ID`: Client ID Pluggy
- `PLUGGY_CLIENT_SECRET`: Secret Pluggy
